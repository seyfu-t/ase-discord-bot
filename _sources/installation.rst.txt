Installation
============

Clone the repository:

.. code-block:: bash

   git clone https://github.com/seyfu-t/ase-discord-bot.git
   cd ase-discord-bot

Install the dependencies using Poetry:

.. code-block:: bash

   poetry install

Alternatively, you can use the provided script to set up a virtual environment:

.. code-block:: bash

   bash scripts/install-deps.sh
   source .venv/bin/activate
