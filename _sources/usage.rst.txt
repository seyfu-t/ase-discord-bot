Usage
=====

Running the Bot Locally
-----------------------

To start the Discord bot, run the following command:

.. code-block:: bash

   bash scripts/run.sh

Available Commands
------------------

- **/recommend_movie**  
  Provides a movie recommendation based on genre, release year (or year range), and language.

- **/recommend_tvshow**  
  Provides a TV show recommendation based on similar parameters.

- **/help**  
  Displays help information with command usage and descriptions.
