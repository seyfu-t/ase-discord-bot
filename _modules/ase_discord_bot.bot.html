ase_discord_bot.bot.discord_bot.run_bot()¶

Run the Discord bot.

This function creates a Discord bot, sets up event handlers and slash commands, and starts the bot using the configured token.

ase_discord_bot.bot.msg_format module¶

ase_discord_bot.bot.msg_format.format_recommendation(results: list[Movie] | list[TVShow]) → list[str]¶

Format a list of media recommendations into displayable strings.

If there are at least three items in the results, randomly selects three to format. Otherwise, formats all provided items.

Parameters:

results (list[Movie] | list[TVShow]) – The list of movies or TV shows to format.

Returns:

A list of formatted recommendation strings.

Return type:

list[str]
